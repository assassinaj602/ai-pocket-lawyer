# Screenshots Folder

This folder contains the app screenshots used in the README.md file.

## Required Screenshots

Please add the following screenshots to this folder:

- `screenshot_home.png` - Main interface for asking legal questions with voice input and image upload
- `screenshot_analysis.png` - Clean, professional AI responses with actionable legal guidance  
- `screenshot_cases.png` - Organize and track your legal cases and evidence
- `screenshot_case_details.png` - Detailed view of case information, deadlines, and attachments
- `screenshot_settings.png` - Configure API keys, themes, and jurisdiction preferences
- `screenshot_document_ocr.png` - Upload and analyze legal documents with OCR technology

## Screenshot Guidelines

- **Resolution**: 1080x1920 (or similar mobile resolution)
- **Format**: PNG for best quality
- **Content**: Show actual app functionality
- **Device**: Use clean device frames or crop to app content only
- **Theme**: Include both light and dark theme examples if applicable

## How to Take Screenshots

1. Run the app on an emulator or physical device
2. Navigate to each screen
3. Take screenshots using device screenshot function
4. Crop and optimize for web display
5. Name files according to the list above
6. Place in this `assets/images/` folder

The README.md will automatically reference these images once they're added.
